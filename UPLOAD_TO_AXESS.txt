QUICK START GUIDE
================================================================================

1. OPEN THE APPLICATION
   - Open index.html in any modern web browser
   - No server required (works offline)

2. ADD PILOTS
   - Navigate to "Pilots" tab
   - Fill in pilot details (name, license, expiry date, etc.)
   - Click "Save Pilot"

3. ADD DRONES
   - Navigate to "Drones" tab
   - Enter drone specifications
   - Assign a pilot
   - Click "Save Drone"

4. SCHEDULE MISSIONS
   - Navigate to "Missions" tab
   - Create new mission with dates and location
   - Assign drone and pilot
   - Click "Schedule Mission"

5. MONITOR FLEET
   - Check map for real-time drone positions
   - View "Analytics" for fleet statistics
   - Check "Alerts" for any critical issues

6. MANAGE MAINTENANCE
   - Schedule preventive maintenance
   - Track maintenance status
   - Assign to technicians
   - Monitor completion

7. MANAGE DOCUMENTS
   - Upload pilot licenses and certifications
   - Set expiry dates
   - Get automatic expiry alerts

8. BACKUP DATA
   - Click "Export" to download fleet data
   - Click "Import" to restore previous data

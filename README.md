FIRST STEPS
──────────────────────────────────────────────────────────────────────────────

✓ View Default Data
  - Navigate to each tab to see sample pilots and drones
  - Check the map to see drone positions
  - View analytics to see fleet statistics

✓ Add Your First Drone
  1. Click "Drones" tab
  2. Fill in drone details:
     - Name: "Your Drone"
     - Model: "DJI Mini 3"
     - Battery: 100
     - Serial Number: "XXXX-XXXX"
  3. Click "Save Drone"
  4. New drone appears on map!

✓ Add Your First Pilot
  1. Click "Pilots" tab
  2. Fill in pilot information:
     - Name: "Your Name"
     - License: "RPL-XXXX"
     - License Expiry: Select a date
     - Contact: "555-1234"
     - Home Base: "Your Location"
  3. Click "Save Pilot"

✓ Assign Pilot to Drone
  1. Go to "Drones" tab
  2. Click "Edit" on a drone
  3. Select pilot from dropdown
  4. Click "Save Drone"

✓ Schedule Your First Mission
  1. Click "Missions" tab
  2. Fill in mission details:
     - Title: "Survey Flight"
     - Select drone and pilot
     - Set start and end times
  3. Click "Schedule Mission"
  4. Mission is tracked in system

✓ Add Maintenance Task
  1. Click "Maintenance" tab
  2. Select a drone
  3. Enter task: "Battery Check"
  4. Set due date
  5. Click "Save Maintenance"

✓ Upload a Document
  1. Click "Documents" tab
  2. Select pilot
  3. Enter document type: "License"
  4. Add reference number
  5. Set expiry date
  6. Optionally upload file
  7. Click "Save Document"

WHAT IF?
──────────────────────────────────────────────────────────────────────────────

Q: My data disappeared!
A: Check browser settings. Allow LocalStorage and try clearing cache.
   Keep your Export backups safe!

Q: The map isn't showing?
A: Requires internet for map tiles. Check connection and refresh.

Q: Can I use on multiple devices?
A: Each browser/device has separate data. Export/Import to sync.

Q: How much data can I store?
A: Typically 5-10MB per browser. For large fleets, use backend.

Q: Is this secure?
A: Data stored locally on your device. No cloud = no external breach risk.

Q: Can I print reports?
A: Export data to JSON and open in spreadsheet or use browser print.
